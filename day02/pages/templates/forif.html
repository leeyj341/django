2. if, else 활용해서 문자열 비교

{% if my_string == my_str %}

{{ my_string }}

{% else %}

간단한 문자열이 아닌데, {{ my_str }}

{% endif %}

3. if, elif, else 사용해보기

{% if my_str|length <= 5 %}

short

{% elif my_str|length >= 10 %}

long

{% else %}

적당

{% endif %}
